# Amrit Raj Portfolio

This is a personal portfolio website for Amrit Raj, a BCA 2nd-year student.

## Files

- `index.html` - main website page
- `styles.css` - website design and responsive layout
- `script.js` - small JavaScript for the current year
- `Amrit_Raj_Web_Developer_Resume.pdf` - downloadable resume

## How to Host on GitHub Pages

1. Go to GitHub and create a new repository.
2. Upload all files from this folder.
3. Make sure `index.html` is in the main/root folder, not inside another folder.
4. Open the repository Settings.
5. Go to Pages.
6. Under Build and deployment, choose Deploy from a branch.
7. Select the `main` branch and `/root` folder.
8. Save.

After a few minutes, GitHub will give you a live website link.
